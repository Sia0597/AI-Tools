# AI-Tools

A collection of Claude Code skills and tools for productivity and automation.

## Skills

| Skill | Description |
|-------|-------------|
| [english-material](skills/english-material.skill) | Generate structured English learning materials from transcripts for Chinese learners |

## Usage

Skills are used with [Claude Code](https://claude.ai/code). Place `.skill` files in your Claude Code skills directory to enable them.
